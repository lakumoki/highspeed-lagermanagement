# Highspeed Lagermanagement

Lagerverwaltungssystem für Highspeed Logistik Schülle Trittau.

## Starten

```bash
npm install
npm start
```

Dann im Browser öffnen: http://localhost:3000

## Login

- **Benutzername:** admin
- **Passwort:** admin

## Funktionen

- Dashboard mit Echtzeit-Statistiken
- Einlagerung mit freien Plätzen
- Paletten-Suche nach EB-Nummer
- Auslagerung mit Lagerlisten-Erstellung
- Visueller Lagerplan (Regale A-F + Blocklager)
- Kundenverwaltung
- Berichte und Druckfunktion
- Löschschutz (Papierkorb + Bestätigungsdialoge)
- Systemprotokoll aller Aktionen
- Benutzerverwaltung mit Rollen

## Lagerstruktur

- Regal A: Platz 1-100
- Regal B: Platz 1-100
- Regal C: Platz 1-100
- Regal D: Platz 1-100
- Regal E: Platz 1-84
- Regal F: Platz 1-103
- Blocklager: Platz 1-600
